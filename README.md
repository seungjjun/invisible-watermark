# Invisible Watermark

파일 그 자체만으로 유출자와 시점을 증명하는 비가시성 워터마크 엔진

## 배경 및 문제 의식

현업에서 상용 비가시성 워터마크 SDK를 사용하며 다음과 같은 한계를 경험

- **Payload 제약**: int형 데이터만 삽입 가능하여 이메일, IP 등 구체적 식별 정보를 담을 수 없음
- **운영 비효율**: 유출 사고 시 유출자 특정을 위해 별도 매핑 테이블 조회 필요

## 프로젝트 목표
> "별도의 DB 조회 없이, 파일 그 자체만으로 유출자와 시점을 증명한다."

- 가변 길이 문자열 Payload 지원 (이메일, IP, 타임스탬프 등)
- 견고성 확보: JPEG 압축, 리사이즈 등 일반적 변환 후에도 검출 가능
- 비가시성 유지: 원본 대비 시각적 품질 저하 최소화
- 처리 성능: 단일 파일 기준 삽입/검출 1초 이내

## 해결 방안

Java 기반 자체 비가시성 워터마크 엔진 구현

- **String Payload 지원**: 사용자 이메일, IP, 타임스탬프 등 가변 길이의 문자열 메타데이터 삽입/검출 기능 구현
- **독립적 검증**: 외부 데이터베이스 의존성을 제거하여 파일 분석만으로 즉각적인 보안 감사 가능

## 기술 스택
- Java 21
- JUnit 5
- Spring Boot 3.5.x
- OpenCV
- Apache PDFBox
